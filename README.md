# Data Delivery Monitoring Platform

## Overview
A Django-based backend application that ingests delivery data, monitors key metrics, 
and flags anomalies to support reporting and issue investigation.

## Motivation
Inspired by real-world experience monitoring data delivery and investigating 
performance issues in production environments.

## Tech Stack
- Python
- Django
- SQL (SQLite/PostgreSQL)
- Linux-compatible setup

## Features
- Data ingestion from CSV
- Metric monitoring and anomaly detection
- Issue logging and reporting dashboard

## What I Learned
- Backend application structure with Django
- Database modelling and SQL queries
- Translating operational requirements into code
- Iterative development and debugging
